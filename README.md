# LocationMax

LocationMax is a location simulation module for rooted Android devices using LSPosed/Xposed.

This public repository is the official distribution and documentation page. It contains release APKs and user-facing information only; the application source code is not published here.

## Installation

1. Download the latest APK from [Releases](../../releases).
2. Install the APK on an Android device with LSPosed/Xposed available.
3. Enable LocationMax in the LSPosed manager and choose only the apps that need location simulation.
4. Open LocationMax to select a location or configure a manual route.
